# AlpinistApp

## Requisitos

* Android Studio instalado.
* Un dispositivo Android físico.
* Cable USB para conectar el dispositivo.
* Opciones de desarrollador habilitadas en el teléfono.
* Depuración USB activada.

## Instalación y ejecución

1. Abrir el proyecto en Android Studio.
2. Conectar el dispositivo Android mediante cable USB.
3. Autorizar la depuración USB cuando aparezca la notificación en el dispositivo.
4. Esperar a que Android Studio sincronice el proyecto y descargue las dependencias necesarias.
5. Seleccionar el dispositivo conectado en la barra superior de Android Studio.
6. Presionar el botón **Run** o utilizar el atajo **Shift + F10**.
7. Esperar a que la aplicación sea compilada e instalada automáticamente en el dispositivo.

## Notas

* Se recomienda contar con conexión a Internet durante la primera ejecución para descargar dependencias de Gradle.
* Si el dispositivo no aparece en Android Studio, verificar que la depuración USB esté habilitada y que el cable permita transferencia de datos.
* La aplicación está diseñada para ejecutarse en dispositivos Android con servicios de ubicación habilitados.
